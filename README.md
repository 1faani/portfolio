## Deskripsi
Project ini merupakan website portfolio pribadi yang dibuat menggunakan
React JS dan Vite. Website berisi informasi profil, tools yang digunakan,
daftar project, serta form kontak.

## Teknologi yang Digunakan
- React JS
- Vite
- JavaScript
- HTML
- CSS
- Tailwind CSS

## Cara Menjalankan Project (Local)
1. Clone repository:
   git clone https://github.com/1faani/portfolio.git

2. Masuk ke folder project:
   cd portfolio

3. Install dependencies:
   npm install

4. Jalankan project:
   npm run dev

## Deployment
Website telah dideploy menggunakan GitHub Pages dan dapat diakses melalui:
https://1faani.github.io/portfolio/
